API Anahtarı Oluşturma

E-Signature Pro API'sini kullanmak için önce bir API anahtarı oluşturmanız gerekiyor. Bu anahtar, tüm API isteklerinizde kullanılacak.

1. API Anahtarı Alma

GET /api/v1/keys
Authorization: Bearer YOUR_TOKEN

Response:
{
  "api_key": "sk_live_abc123...",
  "secret_key": "sk_live_xyz789...",
  "created_at": "2024-01-15T10:30:00Z"
}

2. API Anahtarını Kullanma

POST /api/v1/signatures
Authorization: Bearer sk_live_abc123...
Content-Type: application/json

{
  "document_id": "doc_123",
  "signer_email": "signer@example.com"
}

Güvenlik Uyarısı

API anahtarınızı güvenli tutun ve asla public kodlarda paylaşmayın. Anahtarınızı kaybederseniz hemen yeni bir anahtar oluşturun.

API Endpoint'leri

Doküman İşlemleri

POST /api/v1/documents

Yeni bir doküman yükleyin

POST /api/v1/documents
Authorization: Bearer YOUR_API_KEY

{
  "file": "base64_encoded_file",
  "filename": "contract.pdf",
  "title": "İş Sözleşmesi"
}
GET /api/v1/documents/{id}

Doküman detaylarını alın

GET /api/v1/documents/doc_123
Authorization: Bearer YOUR_API_KEY

İmza İşlemleri

POST /api/v1/signatures

Yeni bir imza talebi oluşturun

POST /api/v1/signatures
Authorization: Bearer YOUR_API_KEY

{
  "document_id": "doc_123",
  "signers": [
    {
      "email": "signer1@example.com",
      "name": "Ahmet Yılmaz"
    }
  ],
  "message": "Lütfen bu belgeyi imzalayın"
}
GET /api/v1/signatures/{id}

İmza durumunu kontrol edin

GET /api/v1/signatures/sig_456
Authorization: Bearer YOUR_API_KEY

Kod Örnekleri

JavaScript

const apiKey = 'sk_live_abc123...';

async function createSignature() {
  const response = await fetch('https://api.e-signature-pro.com/v1/signatures', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      document_id: 'doc_123',
      signer_email: 'signer@example.com'
    })
  });

  const data = await response.json();
  console.log(data);
}

Python

import requests

api_key = 'sk_live_abc123...'
headers = {
  'Authorization': f'Bearer {api_key}',
  'Content-Type': 'application/json'
}

data = {
  'document_id': 'doc_123',
  'signer_email': 'signer@example.com'
}

response = requests.post(
  'https://api.e-signature-pro.com/v1/signatures',
  headers=headers,
  json=data
)

print(response.json())

PHP

$api_key = 'sk_live_abc123...';

$data = [
  'document_id' => 'doc_123',
  'signer_email' => 'signer@example.com'
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, 'https://api.e-signature-pro.com/v1/signatures');
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
  'Authorization: Bearer ' . $api_key,
  'Content-Type: application/json'
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;

cURL

curl -X POST \
  -H "Authorization: Bearer sk_live_abc123..." \
  -H "Content-Type: application/json" \
  -d '{
    "document_id": "doc_123",
    "signer_email": "signer@example.com"
  }' \
  https://api.e-signature-pro.com/v1/signatures
