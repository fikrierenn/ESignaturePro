Başlangıç Rehberi

1

Hesap Oluşturma

E-Signature Pro'ya başlamak için önce hesabınızı oluşturun. E-posta adresinizle kayıt olun ve e-posta doğrulamasını tamamlayın.

POST /api/v1/auth/register
{
  "email": "user@example.com",
  "password": "secure_password",
  "company": "Şirket Adı"
}
2

API Anahtarı Oluşturma

Hesabınızı doğruladıktan sonra API anahtarınızı oluşturun. Bu anahtar ile sistemlerinizi entegre edebilirsiniz.

GET /api/v1/keys
Authorization: Bearer YOUR_TOKEN

Response: {
  "api_key": "sk_live_...",
  "secret_key": "sk_live_..."
}
3

İlk İmza İşlemi

API anahtarınızı aldıktan sonra ilk imza işleminizi gerçekleştirin. Test modunda başlayarak sistemi deneyebilirsiniz.

POST /api/v1/signatures
Authorization: Bearer YOUR_API_KEY
{
  "document_id": "doc_123",
  "signer_email": "signer@example.com",
  "message": "Lütfen imzalayın"
}
